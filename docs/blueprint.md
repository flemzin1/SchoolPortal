# **App Name**: FlemzinPortal

## Core Features:

- Secure Authentication: User authentication system with role-based access control (student&parent to parstud dashboard, staff to staff dashboard, admin to admin dashboard) via registration ID/email and OTP verification.
- Result Viewing: Authenticated students and parents can view academic results by last term only with student reg id.
- Performance Tracking: Visualize academic performance trends over time with charts and graphs.
- School Calendar: Display the school calendar with important dates and events.
- Announcements: Admin and staff can make announcements to parents and students.
- General Support Chat: Allow users to engage in personal and group chats.
- Fee Status: Allow users to view up-to-date fee status and related information.
- Profile: Profile

## Style Guidelines:

- Primary color: Gentle blue (#64B5F6), evoking trust and stability.
- Background color: Very light blue (#E3F2FD), almost white, for a clean and calming interface.
- Accent color: Soft violet (#9575CD), used to highlight interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, is appropriate for both headlines and body text in this MVP.
- Use simple, clear icons that are easy to understand.
- Responsive layout that adapts to different screen sizes and devices.
- Subtle transitions and animations to enhance user experience.
